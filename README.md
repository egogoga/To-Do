# Консольный менеджер задач (To-Do List)

Рождественский Егор МИБ-301
Простой проект на Python для управления задачами с сохранением в JSON-файл.

## Функции

- Просмотр всех задач
- Добавление задачи (с указанием дедлайна)
- Отметка задачи как выполненной
- Удаление задачи
- Автосохранение в файл `tasks.json`

## Как запустить

1. Убедитесь, что установлен Python (версии 3.6+).
2. Склонируйте репозиторий или скачайте файлы.
3. Перейдите в папку с проектом.
4. Запустите: `python todo.py`

## Фича

В проект добавлена возможность указания дедлайна для каждой задачи в формате ГГГГ-ММ-ДД.
